Welcome to MedAE Explorer

MedAE Explorer is a powerful web application designed to help healthcare professionals, researchers, and others easily view and analyze reported medication-related adverse events.

What This Project Is About

Our goal is to provide an intuitive interface for exploring large volumes of adverse event data. You can see details about reported reactions, patients, and related drugs—helping you discover safety signals, uncover trends, and make data-driven decisions.

Key Database Tables

1. Safety_Reports

  • safetyreportid: A unique ID for each adverse event report.
  • serious & other "seriousness" flags: Indicate whether the event is serious, involves hospitalization, etc.
  • receivedate / receiptdate: Dates when the report was first and most recently received.
  • companynumb: Links to a particular company's record.
  • primarysource_reportercountry: Country code for where the report originated.

This table acts as the central source of data for each event—capturing severity, dates, and reporter info.

2. Patients

  • patientagegroup: A coded age group (e.g., 1=Neonate, 5=Adult, etc.).
  • patientsex: 0=unknown, 1=male, 2=female.

Storing basic demographics helps identify any patterns based on patient age or sex.

3. Reactions

  • reactionmeddrapt: Describes the adverse reaction using a standardized MedDRA term.
  • reactionoutcome: Tracks the outcome (e.g., recovered, fatal).

Each safety report can have multiple reactions, detailing the clinical manifestations of the event.

4. Drugs

  • medicinalproduct: The name or brand of the medication.
  • drugcharacterization: Role of the drug in the event (e.g., suspect, concomitant).
  • drugdosageform, drugadministrationroute, drugindication, etc.: Additional context around usage.

This table pinpoints which medications were involved in each report, offering deeper insight into treatment details.

5. Companies

  • companynumb: Unique number identifying the company.
  • companyname: Official name of the manufacturer or organization.

If an adverse event is linked to a specific company, data here helps track and summarize reports by manufacturer.

Application Features

  • All Reports: Quickly see a paginated list of all safety reports and open each for more detail.
  • Drug Search: Find safety reports for a specific drug using an autocomplete-enabled search bar. Narrow your focus and discover relevant cases in seconds.
  • Statistics: Generate charts and data visualizations (e.g., serious vs. non-serious, top reactions), revealing trends and insights at a glance.
  • Admin Panel: A password-protected section for managing the database, allowing authorized users to view, edit, or delete records across all tables.

Ready to Explore?

Use the navigation links above to browse all reports, perform drug-specific searches, or access the statistics dashboard. With MedAE Explorer, discovering meaningful adverse event information has never been easier.
